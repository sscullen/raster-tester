### Setup guide

1. clone git repo

2. Have node 8.9.9 installed

3. run "npm install"

### start tile server first

4. in repo directory, run "npm run tile-server"

### start tester app

5. in repo directory, run "npm run map-server"

### view app

6. go to http://localhost:8081/
